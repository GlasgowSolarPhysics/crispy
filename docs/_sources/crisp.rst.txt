crisp
=====

This submodule contains ``crisp`` which is an object to be used for co-temporal narrow-band CRISP data. This can be used for co-temporal Ca II &lambda;8542 and H&alpha; files and also for either separately. There exists methods to plot spectral lines, Stokes profiles, intensity maps and Stokes maps. This is also used as the base object for the visualisation techniques presented in :doc:`visu`.
